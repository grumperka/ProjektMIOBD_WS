By zarejestrować mikroserwis w EURECA należy:

1. W środowisku Intellij otworzyć projekt z folderu "service-registration-and-discovery-service".
2. Uruchomić powyższy projekt i przejść pod adres:

	http://localhost:8761/

4. Nie zamykamy powyższej strony.
3. W środowisku Visual Studio otwieramy projekt z folderu "ProjectMongoDBReact". Jeżeli chcemy użyć na tym etapie aplikacji hotelu, powinniśmy zainicjalizować bazę danych w MONGODB (wszystko jest opisane w pliku READme.txt).
4. Uruchamiamy projekt.
5. Gdy projekt w Visual Studio jest już uruchomiony, odświeżamy stronę EUREKA.
6. Nasz mikroserwis powinien być na liście zarejestrowanych serwisów.
